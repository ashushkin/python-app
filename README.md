# python-app

Для запуска приложения:

For linux:

1 - python3 -m venv <DIR>
2 - source <DIR>/bin/activate
3 - pip3 install -r requirements.txt
4 - python3 main.py

For windows:

1 - py -m venv <DIR>
2 - <DIR>\Scripts\activate
3 - pip3 install -r requirements.txt
3 - py main.py
